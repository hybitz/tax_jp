# TaxJp

税金計算ライブラリ
* 消費税
* 社会保険（健康保険・厚生年金）
* 雇用保険
* 源泉徴収
* 減価償却率

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tax_jp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tax_jp
