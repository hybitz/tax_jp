# TaxJp

税金計算ライブラリ
* 消費税
* 社会保険（健康保険・厚生年金）
* 雇用保険
* 源泉徴収
* 減価償却率

## インストール

Gemfile に以下の記述を追加

```ruby
gem 'tax_jp'
```

bundle を実行

    $ bundle

## 使い方

### Rails-5.2

routes.rb に以下の記述を追加

```ruby
  mount TaxJp::Engine, at: '/tax_jp', as: 'tax_jp'
```

Railsを起動

    $ rails s

ブラウザで localhost:3000/tax_jp にアクセス
